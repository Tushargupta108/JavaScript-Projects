# ❤️ Love Calculator

A fun and simple Love Calculator built using HTML, CSS, and JavaScript.

## 🚀 Features

- Takes two names as input
- Calculates a fun love percentage
- Displays result instantly without page refresh
- Responsive and modern UI
- Beginner-friendly JavaScript project

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## 🧠 Logic Used

```javascript
const result = Math.pow(
  boy.value.length + girl.value.length,
  3

#The formula:
Find the length of both names.
Add the lengths.
Raise the sum to the power of 3.
Take modulo 101 to get a percentage between 0 and 100.
