# 🔮 Astrology Insights

Astrology Insights is a fun and interactive web application built using **HTML, CSS, and JavaScript**. The app generates personalized astrology-themed insights based on the user's name and date of birth.

It combines zodiac signs, compliments, personality observations, Vastu-inspired recommendations, and random predictions to create an engaging and entertaining experience.

---

## ✨ Features

* ♈ Displays the user's zodiac sign based on the birth month.
* 💫 Generates unique compliments based on the birth date.
* 🌟 Provides personality-style insights.
* 🏡 Suggests Vastu-inspired recommendations.
* 🔮 Gives random astrology predictions.
* 📱 Fully responsive design for desktop and mobile devices.
* 🎨 Modern glassmorphism-inspired user interface.

---

## 🛠️ Tech Stack

* **HTML5** – Structure and form handling
* **CSS3** – Responsive design and styling
* **JavaScript (ES6)** – Logic, arrays, DOM manipulation, and dynamic content generation

## 🚀 How It Works

1. Enter your:

   * First Name
   * Surname
   * Date of Birth
   * Birth Month
   * Birth Year

2. Click **GET INSIGHTS**.

3. The application generates:

   * Your Zodiac Sign
   * A Personalized Compliment
   * A Personality Insight
   * A Vastu Recommendation
   * A Fun Astrology Prediction
